What is ia responsive site, and why is it important?

When using websites on different devices, like smart phones, tablets or desktops, the content needs to adjust to the size of he screen it is being displayed on. This is where responsive sites come in. Adding in extra code to the site so that no matter the size of the screen the site is always going to be understandable and well presented.
This is important because there is such a wide variety of devices to use the internet on and all are becoming more popular and more used, so sites must keep up with change.

What is mobile design, and why is it important?

Mobile design, is designing your site for a smaller screen first and then adjusting the code so that as the screen gets bigger the site adjusts as well to keep it lookinf normal.
It is important because it is a lot easier to make a site grow than shrink.

What are frameworks, and what are their pros and cons?

A framework is defined as a package made up of a structure of files and folders of standardized code (HTML, CSS, JS documents etc.) which can be used to support the development of websites, as a basis to start building a site.
Pros and Cons of using Frameworksstrong>
Pros

  • Speeds up the mock-up process
  • Clean and tidy code
  • Browser compatibility
  • Helpful in collaborative work
  • Learn good practices

Cons
  • Slower learning curve
  • Unused code left over
  • Mixes content and presentation
  • You don't learn to do it yourself

What is a wireframe, and why do we use it?

A wireframe is like a plan of your website, drawn up on paper or using a graphics tool. This helps to code the site with less time wasted, as everything is already on paper and just needs coding.
